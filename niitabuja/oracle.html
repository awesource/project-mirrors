Oracle 11g & 12c Administration

Database Administrators (DBA) are responsible for the design, implmentation, support and maintenance of computarised databases in today's organisations. The role also includes architecting, building and scaling databases for future data growth adn capscity. They are also responsible for security, performance availability of data to users and customers. All the above tasks are perfomed with the help of a Database Management System (DBMS) and the leading and most widely used popular and widely used database in the world todya. This course is designed to equip participants with the necessary skills to successfully manage and adminsteran Oracle database while adhering stricly to best practices.

Course Objectives

  • Store, retrieve and modify data in tha database using SQL queries
  • Understand Oracle database logical and physical storage structures
  • Administer users and security
  • Install, create and adminster Oracle Database 11g Release 2
  • Configure the database for an application
  • Employ basic monitoring procedures
  • Implementing a backup and recovery strategy
  • Move data between databases and files
  • Tune and Oracle Database for optimal performance
  • Configure Oracle Database for optimal recovery
  • Backup and recover a database (and its parts) with recovery manager (RMAN)
  • Use flash backup technology to view past state of data and to recover objects to a past state
  • Identify burdensome database sessions and poorly perfoming SQL queries
  • Use and appropriate and flexible memory configuration
  • Configure resource allocations accross sessions and tasks
  • Schedule jobs to run inside and outside the database
  • Use compression to optimse database storage and duplicate a database
