Our Services

Advanced Excel Tools & Techniques For Data Analysis

Microsoft Excel is one of the most widely used solutions for analysing and visualising data. Beginning with Excel 2010, new tools were introduced to enable the analysis of more data resulting in less time spent creating and maintaining the solutions and enabling a better understanding for what the data means. This better understanding is facilitated by improved visualisations and more sophisticated business logic

Course Objectives

At the end of the programme, participants would be able to:

  • Use the advanced functions in Excel
  • Reference data effectively
  • Create reports using Pivot table and Pivot chart
  • Use Macro to automate tasks
  • Create scenarios and What-if-analysis
  • Use Conditional formatting techniques
  • Understand workbook security and troubleshoot invalid data
  • Establish trends in data series

Course Outline

  • Calculating data across multiple worksheet
  • Analysing data with Logical and Lookup functions
  • Using Text functions to generate reports
  • Using Statistical and Financial functions
  • Creating reports using PivotTable and PivotCharts
  • Working with Macro
  • Applying conditional format to data
  • Creating Trendlines and Sparklines
  • Creating Scenarios
  • Performing What-if-analysis
  • Managing workbook security
  • Troubleshooting formulas and invalid data
